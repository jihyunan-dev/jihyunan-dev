## 소개
안녕하세요, 3년차 프론트엔드 개발자 안지현입니다.   
저는 사람들이 효율적으로 일하게 하는 환경을 만드는 것에 관심이 많습니다.   
회사 내에서는 어드민을 가다듬고, 디자인 시스템을 만들며 동료 각각이 더 중요한 일에 집중할 수 있도록 돕습니다.   
회사 밖으로도 모두가 더 자신의 커리어와 집중해야 할 것에 시간을 쏟을 수 있는 도구를 만드는 것을 좋아합니다.
